Como Inserir Imagens Usando HTML

Adicionar imagens a um site ou perfil de alguma rede social é uma excelente maneira de melhorar o aspecto visual da página. O código HTML (Linguagem de Marcação de Hipertexto) para adicionar imagens é bastante simples e geralmente é o primeiro a ser aprendido por um estudante de HTML.

  1. Salve a imagem no diretório do projeto. Atualmente, existem diversos serviços gratuitos de hospedagem de imagens, como o Picasa Web Albums, Imgur, Flickr ou o Photobucket.
  2. Abra seu arquivo HTML. Abra o documento HTML correspondente à página web na qual a imagem será exibida.
  3. Comece com a tab IMG. Encontre o local no corpo do arquivo HTML no qual você deseja inserir uma imagem. Escreva a tag <img/>
  4. Encontre o endereço URL da sua imagem. Visite a página da internet na qual você hospedou a imagem.
  5. Coloque o endereço da URL dentro do atributo SRC.
  6. Salve as alterações. Salve o arquivo HTML do site.
